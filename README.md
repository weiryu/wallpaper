## 功能简介

* 可选择bing每日精选图片，国家地理每日精选图片和wallheaven每小时精选图片自动设置为桌面壁纸
* 支持指定本地文件设置为壁纸，支持壁纸删除功能
* exe文件同级目录下生成txt文件用于存放壁纸保存位置信息
* 不同模块壁纸分文件夹存放，便于管理
